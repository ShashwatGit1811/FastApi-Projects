# 🚀 FastAPI Projects

A collection of beginner to intermediate **FastAPI** projects built while learning backend development with Python. This repository contains practical projects that cover core FastAPI concepts, REST APIs, validation, authentication, database integration, and CRUD operations.

📚 What You'll Find

- User Registration & Login APIs
- CRUD Operations
- Request Validation using Pydantic
- Query & Path Parameters
- Error Handling
- Authentication Basics
- MySQL Database Integration
- RESTful API Design
- Interactive API Documentation with Swagger UI

- FastAPI

- Uvicorn

- MySQL Connector/Python

▶️ Running a Project

1. Clone the repository
   ```bash
   git clone <repository-url>
   ```

2. Navigate to the project folder
   ```bash
   cd project-folder
   ```

3. Install dependencies
   ```bash
   pip install -r requirements.txt
   ```

4. Start the server
   ```bash
   uvicorn main:app --reload
   ```

5. Open Swagger UI
   ```
   http://127.0.0.1:8000/docs
   ```
